# v0.38 Home Meta Progression Design

## Goal

Turn the current home return loop into a small roguelike meta-progression loop. A finished run should bring back a resource, the player should spend it in the home scene, and the next run should start slightly stronger.

## Scope

This version adds one resource and three permanent upgrades. It does not add a full shop, multiple save slots, skill trees, new UI prefabs, or narrative unlocks.

## Resource

The resource is `Core Fragments`. Runs award fragments when they end:

- cleared rooms give the base reward;
- victory gives a bonus;
- active growth-route level and route-special rewards add small bonuses.

The earned amount and current total are stored in `RunSessionSummary` so both the battle completion panel and home summary can display them.

## Upgrades

The home scene has a simple `HomeMetaUpgradeStation`. When the player stands near it, the station shows current fragments and three purchase options:

- Player Max HP;
- Player Damage;
- AI Support Cooldown.

Each upgrade has an increasing cost based on its current level. Purchases spend fragments, raise the level, and trigger autosave.

## Save Integration

`GameSaveData` gains a `metaProgression` section. The existing `GameSaveService` loads and saves both relationship state and meta-progression state. The current autosave session listens to both relationship changes and meta-progression changes.

Old saves remain valid because missing `metaProgression` data normalizes to an empty progression state.

## Next Run Effects

`RunManager.StartRun` applies permanent upgrades to scene objects before the first room starts:

- Max HP level increases player max health and refills it;
- Damage level multiplies player base attack;
- AI cooldown level reduces companion attack cooldown.

The effects are intentionally modest so the demo remains readable.

## Testing

EditMode tests cover:

- save round-trip for fragments and upgrade levels;
- run end awards fragments and records summary lines;
- purchasing an upgrade spends fragments and raises the right level;
- starting a run applies the permanent upgrades to player and companion stats.
