# v0.33 Counterplay Reward Link Design

## Goal

Turn the v0.32 player counterplay loop into a small growth path. Room rewards should be able to strengthen dash dodges, recovery protection, dodge counterattacks, and Guard opening punishment.

## Scope

- Add counterplay reward types to the existing room reward pool.
- Let dash cooldown, recovery window, dodge counter damage, and Guard opening damage grow independently.
- Apply counterplay damage bonuses during normal player attack resolution.
- Show current counterplay growth and temporary dodge boost state in the existing status panel.
- Let the AI companion acknowledge counterplay reward picks through the existing speech bubble system.

## Out of Scope

- No permanent skill tree UI.
- No save schema expansion for counterplay upgrades.
- No new input bindings.
- No animation controller rewrite.
- No new enemy archetype.

## Architecture

`RunManager` owns reward selection and applies reward effects. v0.33 adds four reward types: dash cooldown, recovery window, dodge damage boost, and Guard opening damage. The candidate list appends these rewards at runtime so existing scenes remain playable even if their serialized reward arrays are older.

`PlayerMovement2D` exposes dash cooldown as a readable value and a multiplier method. `PlayerCounterplayFeedback` becomes the counterplay growth state holder. It tracks recovery duration, dodge boost duration/multiplier, Guard opening multiplier, and the active dodge boost timer.

`PlayerCombat2D` asks `PlayerCounterplayFeedback` to modify outgoing player damage per target before `HealthComponent.TakeDamage`. This keeps normal player damage rewards and branch buffs intact while allowing counterplay-specific damage to stack.

`StatusFeedbackUI` reads the player's counterplay component and displays a compact HUD line, including permanent upgrade values and active dodge boost time.

## User Feedback

- Reward choices should visibly use the "Counterplay" label.
- Choosing a counterplay reward should trigger a short AI speech bubble.
- The status panel should make it clear when a dodge counter boost is currently active.
- Guard opening hits should keep the existing v0.32 feedback but now benefit from the upgraded damage multiplier.

## Testing

EditMode tests cover:

- Counterplay rewards expose readable reward choices.
- Dash cooldown reward reduces player dash cooldown.
- Recovery reward extends follow-up protection.
- Dodge reward boosts the next player attack after a projectile dodge.
- Guard opening reward increases damage against a vulnerable Guard.
