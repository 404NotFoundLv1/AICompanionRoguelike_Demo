# v0.32 Player Counterplay Feedback Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a small playable player counterplay loop around dash invulnerability, post-hit protection, projectile dodging, Guard opening feedback, and AI tactical callouts.

**Architecture:** Add one focused player component, `PlayerCounterplayFeedback`, that implements `IDamageModifier` and subscribes to existing player movement/combat events. Keep enemy attacks routed through `HealthComponent.TakeDamage`; only adjust projectile resolution so canceled damage counts as a dodge/miss. Use existing companion speech bubble hooks for tactical hints.

**Tech Stack:** Unity 6000, C#, NUnit EditMode tests, Unity MCP verification.
