# v0.38 Home Meta Progression Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a small persistent home meta-progression loop where run endings award Core Fragments, home upgrades spend them, and the next run starts stronger.

**Architecture:** Add a focused `MetaProgressionState` runtime state object and save DTOs under the existing save pipeline. Let `RunSessionState` award fragments on run end, `RunManager` apply saved upgrades at run start, and a small home station component expose purchases in the home scene.

**Tech Stack:** Unity 6000, C#, Unity `JsonUtility`, NUnit EditMode tests, Unity MCP scene setup.
